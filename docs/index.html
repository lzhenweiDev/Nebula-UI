Introduction

Nebula UI v3.1 is a modern, customizable UI library for Roblox with a glossy purple theme, smooth animations, notifications, loading screens, and many built-in components.

🎨 Amethyst / Light 📂 Tabs & Sections 🔘 Toggle · Button · Slider 📋 Dropdown · TextBox 🔑 Keybind · Checkbox 🎨 RGB Color Picker 📊 Progress Bar 🔔 Notifications ⏳ Loading Screen 💾 Config System 🖱 Draggable Window ✨ Glossy Effect

The library uses V3.1 with improved performance and new features.

Installation

Load the library with a single line:

local Nebula = loadstring(game:HttpGet("https://github.com/lzhenweiDev/Nebula-UI/raw/refs/heads/main/V3.lua"))()

The current version is V3.1. For modifications, use source.lua.

Window

Create a main window with configuration and optional loading screen.

local Window = Nebula.CreateWindow({
    Title = "Nebula Demo",
    ConfigName = "MyConfig",
    ShowLoading = true,
    LoadingDuration = 3,
    LoadingSteps = {
        "Loading...",
        "Creating UI...",
        "Almost Done...",
        "Ready!"
    }
})
ParameterDescription
TitleWindow title
ConfigNameName for the config system
ShowLoadingShow loading screen (true/false)
LoadingDurationDuration in seconds
LoadingStepsArray of status messages

Methods: :Destroy(), :Minimize(), :Notify()

Tabs & Sections

Organize your UI with tabs and subsections.

-- Create tabs
local MainTab = Window:CreateTab("Main", "🏠")
local SettingsTab = Window:CreateTab("Settings", "⚙")

-- Section with title
local MainSection = Nebula.CreateSection(MainTab, {
    Name = "Main Features"
})

-- Section without title
local EmptySection = Nebula.CreateSection(MainTab)
ParameterDescription
Name (Tab)Tab name
Icon (Tab)Emoji or icon
Name (Section)Optional section title

Button

A simple button with callback and optional color.

MainSection:CreateButton({
    Name = "Click Me",
    Color = Color3.fromRGB(255, 0, 0), -- optional
    Callback = function()
        print("Clicked!")
    end
})

Toggle

A switch with state, events, and methods.

local Toggle = MainSection:CreateToggle({
    Name = "Auto Farm",
    Default = false,
    Callback = function(Value)
        print(Value)
    end
})

-- Methods
Toggle:Set(true)
print(Toggle:Get())

-- Event
Toggle.Events.OnChanged:Connect(function(Value)
    print("Changed:", Value)
end)

Slider

Value range with min, max, suffix, and input field.

local Slider = MainSection:CreateSlider({
    Name = "WalkSpeed",
    Min = 16,
    Max = 100,
    Default = 16,
    Suffix = " Studs",
    Callback = function(Value) end
})

Slider:Set(50)
print(Slider:Get())
Slider.Events.OnChanged:Connect(function(v) end)

Dropdown

Select from a list of options with refresh function.

local Dropdown = MainSection:CreateDropdown({
    Name = "Select Team",
    Options = { "Red", "Blue", "Green" },
    Default = "Red",
    Callback = function(Value) end
})

-- Refresh options
Dropdown:Refresh({ "Option 1", "Option 2", "Option 3" })
Dropdown:Set("Blue")
Dropdown.Events.OnChanged:Connect(function(v) end)

TextBox

Text input with placeholder and events.

local Box = MainSection:CreateTextBox({
    Placeholder = "Type here...",
    Default = "",
    Callback = function(Text) end
})

Box:Set("Hello")
print(Box:Get())
Box.Events.OnChanged:Connect(function(Text) end)
Box.Events.OnFocus:Connect(function() end)
Box.Events.OnBlur:Connect(function() end)

Progress Bar

Progress bar with current value and optional color.

local Progress = MainSection:CreateProgressBar({
    Name = "Loading",
    Value = 30,
    Max = 100,
    Color = Color3.fromRGB(140, 80, 255) -- optional
})

Progress:Set(75)
print(Progress:Get())

Notifications

Notifications with type, duration, and icon.

Window:Notify({
    Title = "Success",
    Content = "Everything loaded!",
    Type = "success", -- info | success | warning | error
    Duration = 5,
    Icon = "✅"
})
TypeDescription
infoInformation (accent color)
successSuccess (green)
warningWarning (yellow)
errorError (red)

Loading Screen

Automatically enabled via ShowLoading in the Window config.

local Window = Nebula.CreateWindow({
    Title = "My App",
    ShowLoading = true,
    LoadingDuration = 3,
    LoadingSteps = {
        "Initializing...",
        "Loading Modules...",
        "Ready!"
    }
})

The loading screen shows a progress bar with percentage and step messages.

Config System

Save, load, and delete configurations (supports writefile).

-- Save
ConfigManager:Save("MyConfig", data)

-- Load
local data = ConfigManager:Load("MyConfig")

-- Delete
ConfigManager:Delete("MyConfig")

Themes

Two built-in themes: Amethyst (default) and Light.

-- The theme is set when creating the window
-- Default: Amethyst (purple glossy)
-- Light: Bright theme for daytime use

Amethyst is the default theme with Glossy Effect. Light theme has the glossy effect disabled.
